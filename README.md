任务二
1.我下载了群文件里的 `task2.csv`，用 pandas 读取后发现数据有两列：x 和 y。

2.模型构建
我搭建了一个3层的全连接神经网络（MLP）：
- 输入层：1个神经元（接收x）
- 隐藏层：10个神经元，激活函数用ReLU
- 输出层：1个神经元（输出预测的y）
损失函数用 MSELoss（均方误差），优化器用 Adam，学习率先设为0.01。

3.训练过程
训练了1000轮，Loss从167.21逐渐下降到161.88，说明模型在学习。

4.学习率分析
考核要求我们尝试不同的学习率，我做了三组实验：

实验1：学习率=1.0（太大）
- 现象：Loss一直震荡在162.7左右，降不下去
- 原因：学习率太大，参数更新步子迈太大，直接跳过了最优点

实验2：学习率=0.01（正常）
- 现象：Loss平稳下降，最终到161.88
- 原因：学习率适中，稳步收敛

实验3：学习率=0.00001（太小）
- 现象：Loss下降非常慢，1000轮只降到191.95
- 原因：学习率太小，参数更新步子太小，走不动

5. 过拟合判定
考核还要求我们观察过拟合现象，我做了两组对比：

正常模型（未过拟合）
- 网络：3层 (1-10-1)
- 轮数：1000
- 现象：Loss平稳下降，拟合曲线光滑，符合数据趋势

过拟合模型
- 网络：加深到5层 (1-100-100-100-1)
- 轮数：10000
- 现象：
  - Loss降到了0.5以下（几乎为0）
  - 拟合曲线剧烈扭曲，试图穿过每一个数据点
- 判断依据：训练Loss极低，但拟合曲线不光滑，说明模型把噪声也学进去了
